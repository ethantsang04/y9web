Inquiry and Analysis - Basic Sanitation Services

Explanation & Justification

In more modern times, sanitation has become vital in our society. It is a necessity to be safe from diseases and prevent the transmission of diseases. As people become more aware of the importance of sanitation, naturally, the attempt to share this information becomes more frequent. Websites such as the World Health Organization (WHO) use charts and tabular data to display information about topics such as sanitation. However, these tables are very basic, displaying only a mere skeleton of data. Furthermore, they are not at all aesthetically pleasing to the eye, thus people tend to lose interest very quickly. Nowadays, people are drawn to more modern websites that use ergonomic websites that are user-friendly, simple and aesthetically appealing. Therefore, in order to share this vital information, one must transform the boring tabular data found on the WHO website and transform it into a website that can display data in a ergoomic fashion. So, it is thusly justified to develop a website using HTML that looks good, and shows sufficient data using graphs and charts rather than tabular data that can entice users to continue to read about this topic and become more aware of it, which in turn, will overall benefit those in need of basic sanitation services.

Primary Research / Skills

In order to achieve this type of website, a couple of basic skills are required. Basic HTML, CSS and JavaScript should all be used to create the best possible website to display this data. In order to create the "bare bones" of the website, HTML must be used to develop well-written and descriptive paragraphs the convey the data in an interesting way. However, sentences can not run on for too long, otherwise I might lose the interest of the user. This is the most important aspect of the website. In addition, CSS should be used to make the website look aesthetically pleasing. CSS is usually yu rfirst impression of a website. If it looks professional, people tend to stay and explore the page a little longer. On top of that, JavaScript must be utilised to add interactive functions. For this project, HTML and CSS are probably the more important languages when creating a solution to this problem. These are the languages that are essential to displaying data graphically. In addition, they compliment each other very well. CSS can be used to insert graphs, charts, etc, and HTML can inout raw data into these graphs and charts. Using CSS to create ways to display data graphically is another skill set that I will need to achieve in order to complete this project, along with some simple HTML to create the best website possible.
